# 今日小账真实记账增强计划

## 目标

把今日小账从基础记账原型升级为更接近真实记账软件的本地 App。

核心优先级：

1. 记账必须方便、真实、可回看。
2. 预算、账户、待结清事项要成为账本能力的一部分。
3. 角色评账只能读取真实数据，不能虚构账单以外的信息。

## 已规划能力

- 默认进入记账页。
- 备注关键词自动分类。
- 同日相似账目重复提醒。
- 本月总预算和分类预算。
- 微信、银行卡、现金、其他账户。
- 转账与手续费。
- 待结清事项：预付、报销、医保、自费、次数卡、其他。
- 明细页展示收入、支出、结余、预算剩余和待结清摘要。
- AI 短评和月总结读取真实统计。

## 文件分层

- `src/domain/` 写纯账本规则。
- `src/app/` 写界面渲染和交互调度。
- `src/styles/` 写主题和组件样式。
- `tests/` 写规则和架构测试。
- `docs/` 写产品说明和开发计划。

## 非目标

- 不做云同步。
- 不做复杂资产负债表。
- 不做贷款完整摊销。
- 不让 AI 决定基础记账分类或金额。
- 不自动生成不存在的账目。
